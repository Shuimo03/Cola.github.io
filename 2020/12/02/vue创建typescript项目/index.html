vue创建typescript项目

根据官方步骤来先安装vue-cli3脚手架，可以根据之前的文章来看是如何安装的，安装成功之后，在命令行中，
输入vue create project-name(项目名)，接着会显示vue-cli的版本号和一些选项，因为是要安装typescript的项目，所以选择 manually select features，按下enter进入后，选中typescript和一些常用常见，通过方向键+空格。

选中之后，会让选中vue.js的版本，这里选中默认的2.x，选中之后，到了询问是否使用 class风格的组件语法，为了方便使用typescript，就选择Y。
