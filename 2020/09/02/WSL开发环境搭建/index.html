WSL开发环境搭建

软件列表

  1. golang
  2. scrapy

golang

因为某些特殊原因，需要使用代理才能下载，所以可以现在宿主机下载好以后，在从wsl2中复制过去，具体步骤就不写了，当然，如果你已经能够使用代理了，就可以使用如下命令。

安装过程

如果没有代理就要输入这些命令

当把安装包移动到指定目录的时候，就可以开始配置golang开发环境了。

最后运行 ~/.bashrc，为了确认有没有安装成功，输入go version，如果出现版本号，就说明安装成功了。

安装scrapy

首先要安装pip3或者pip，这里讲下怎么安装pip3，输入命令

sudo apt-get install python3-pip

安装好后，接着输入以下命令

  1. sudo apt-get install python-dev
  2. sudo pip install scrapy

最后等进度条跑完就行了。

结语

完全可以当作Ubuntu来用，包括安装软件。
