WSL2开发配置

安装WSL2

首先Windows10的版本必须是1903或者更高的版本才行。然后用管理员身份打开powershell，然后运行下面命令。

设置WSL2为默认版本

选择要安装的Linux版本

打开Microsoft Store，在里面搜索要安装的Linux版本，如下

我自己安装的是Ubuntu20.04，安装好以后就可以在powershell里面，输入ubuntu就启动了，然后就和安装Ubuntu一样，一步步设置过来。

安装zsh和ohmyzsh

zsh是一个shell工具，要比Ubuntu自带的bash好用，可以参考知乎上这个回答

https://www.zhihu.com/question/21418449

接着安装oh my zsh，oh my zsh 可以让zsh变得更好用，更香，如果你有梯子的话可以使用下面命令安装

没有的话，就要GitHub上面把他clone下来。

接着cd ohmyzsh/tools/，然后./install.sh

然后安装两个插件

  • zsh-syntax-higlighting语法高亮插件

    • git clone https://github.com/zsh-users/zsh-syntax-highlighting.git
